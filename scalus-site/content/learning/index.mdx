## Design Patterns

Proven patterns for building efficient and secure Cardano smart contracts:

- **[Stake Validator Pattern](/docs/learning/design-patterns/stake-validator-pattern)** - Delegate computation to staking scripts for multi-input optimization
- **[Transaction Level Minter](/docs/learning/design-patterns/transaction-level-minter-pattern)** - Couple spend and mint endpoints for minimal spend costs
- **[UTxO Indexer](/docs/learning/design-patterns/utxo-indexer-pattern)** - Match inputs to outputs using off-chain computed indices
- **[Validity Range Normalization](/docs/learning/design-patterns/normalized-interval-pattern)** - Simplify time range validation with normalized intervals
